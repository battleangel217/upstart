## Getting Started

1. Open `index.html` in your browser to access the marketplace
2. Sign up as a customer or vendor at `signup.html`
3. Login at `login.html` to start buying or selling

## Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Python (Django/FastAPI)
- **Real-time**: WebSockets
- **Payments**: Paystack
- **Storage**: Supabase
- **Authentication**: JWT tokens

## Pages

- `index.html` - Main marketplace
- `login.html` / `signup.html` - Authentication
- `profile.html` - User profile management
- `cart.html` - Shopping cart
- `wallet.html` - Digital wallet
- `chat.html` - Messaging
- `inventory.html` - Vendor inventory (vendor-only)
- `analytics.html` - Sales analytics (vendor-only)
- `leaderboard.html` - Community rankings
- `vendor-profile.html` - Public vendor profiles
